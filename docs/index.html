Plugins

Each plugin adds one or more slash commands to your agent.

ship
Git add, commit, and push in one step
pr
Create a GitHub PR with auto-generated description
tech-writer
Review and rewrite docs via Google's style guidelines
deck
Generate a self-contained HTML slide deck
roadmap
Visual HTML Gantt chart from a markdown file
diagram
Generate interactive SVG diagrams from a description

Install

Choose your agent.

Headless (one-liner)

Pipe the installer directly into codex exec — no interaction required.

bash 
curl -sL https://raw.githubusercontent.com/mikersays/mikersays-plugins/master/INSTALL.md \
  | codex exec --full-auto -

Uses --full-auto so the agent applies all steps without prompting. Safe to re-run — it detects existing installs and skips them.

Interactive

Open a Codex session in any directory
bash 
codex
Paste the install prompt
The agent will clone the repo, register all plugins, and install the auto-update hook.
prompt 
Read @INSTALL.md and follow the instructions exactly to install the
mikersays-plugins marketplace on this machine. Create or update the
local marketplace file and hooks config, verify the install, and tell
me the final result.

What gets installed

Repo cloned to ~/.codex/plugins/mikersays/mikersays-plugins
All 6 plugins registered in ~/.agents/plugins/marketplace.json
SessionStart hook added to ~/.codex/hooks.json — auto-pulls on every Codex startup

Add the marketplace

slash command 
/plugin marketplace add mikersays/mikersays-plugins

Install individual plugins

slash command 
# Install any plugin by name
/plugin install ship@mikersays-plugins
/plugin install pr@mikersays-plugins
/plugin install tech-writer@mikersays-plugins
/plugin install deck@mikersays-plugins
/plugin install roadmap@mikersays-plugins
/plugin install diagram@mikersays-plugins

examples 
# Commit and push all changes
