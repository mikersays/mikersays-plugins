What gets installed

Repo cloned to ~/.codex/plugins/mikersays/mikersays-plugins
Selected plugins registered in ~/.agents/plugins/marketplace.json
Skills symlinked into ~/.agents/skills/ — appear in /skills
SessionStart hook added to ~/.codex/hooks.json — auto-pulls on every startup
